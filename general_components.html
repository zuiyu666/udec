Common Commands and FISH

The model objects described in this section are not specific to any one program but are commonly available to each of the following Itasca programs: PFC2D, PFC3D, FLAC3D, UDEC, and 3DEC — hence the appellation “common.” This means that there is a significant amount of knowledge transferability between these programs. Grasp of the way groups work in UDEC, for instance, is sufficient to confer an understanding of how they will work in UDEC. In any of these programs the objects found here achieve the same kinds of purposes, have the same underlying logic, and operate under the same assumptions.

This documentation, therefore, is split such that everything that is common to the programs is contained in the section Program Guide, and information on commands and FISH specific to UDEC are in the section that follows.
