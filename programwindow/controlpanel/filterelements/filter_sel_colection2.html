UDEC

SEL Collection

Usage

This filter element can be used to filter a list of items by structural element collection ID number. By default, it appears with the “Not” attribute set.

See below for further discussion.

Interactive Operation

Position the mouse over the plot item object of interest. When the cursor indicates remove mode is ready ( ), click on the object to be removed. Repeat as needed. Note that each removal click will add an item (identified by Material number) to the “Add” attribute. Items added to the “Add ” list cannot be removed, but the check boxes next to their names may be unchecked to “un-include” them in the filter. Assuming the user is interested in filtering the plot item to only display those items selected, the last step in using this element is to uncheck the “Not” attribute.

Attributes

Not: Invert the filter element’s selection.

Add: Specify the list of Material numbers to include in the filter. Values may be typed into the field adjacent to the “Add” label and added to the list by pressing Enter. The check box next to each added object ID may be used to include (checked) or exclude (unchecked) the object from the action of the filter.
