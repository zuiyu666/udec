Tools Menu

Figure 1: The Tools menu.

Options
Open the Options dialog.
Disable Tooltips
Turn off display of tips that appears when the mouse hovers over interface tools (e.g., buttons on a toolbar, dialog controls, etc.)
Bundle
Open a fly-out menu containing the choices “Pack…” and “Unpack….” The former allows the current project to be packed and saved as a bundle file; the latter calls a dialog that allows the user to pick a bundle file to unpack and open in UDEC. See the topic “Project Bundle Files” for more information on this feature.
Copy App Data…
Open a dialog that allows the user to create a user-accessible folder for all the examples and other ancillary material that comes with UDEC (the material is copied to a user-specified location from the system-protected area where it is initially written on installation of the program).
Check Modifications…
Open a browser window that accesses the Itasca website to display the latest modifications to the program.
Plugins…
Open a dialog that shows the plugins currently loaded in UDEC.
Update key…
Open a dialog that allows the user to update their hardware lock (security USB key) when the purchase of an upgrade, option, or other program addition has caused a change to the license that must be recognized by the key. This action requires the user to have a valid password file (supplied by Itasca at the time of license modification) in order to the complete the key update process.
