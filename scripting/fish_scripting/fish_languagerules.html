FISH Rules and Usage

Information in this section covers the foundational aspects of the FISH language. The FISH Function Index in this section indexes the built-in FISH functions. The FISH functions available to query and manipulate UDEC model components (e.g., blocks, zones, contacts) are available in the “UDEC Model Components” section. Functions for use with the general components of Itasca software are cataloged in the “FISH Summary section.”
