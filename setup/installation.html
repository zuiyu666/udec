UDEC

Installation

The program is installed via a Microsoft Software Installation (.msi) file. The installation is typically delivered via a USB drive (on a standard or network license purchase), but, in some cases, may also be acquired via download. The program is designed to run on Microsoft Windows systems (Windows 7, Windows 8, and Windows 10 are currently supported).

The default installation of will install the program, its example files and the complete documentation. This document set, which contains the full program documentation, may be accessed via the Help —> Help menu option in the program.

Run the Installation

The program is installed from a software installation provided via USB drive, DVD or download. To install using a USB drive or DVD, insert the installation media in the computer. Once inserted, an Autoplay dialog will appear. If using a flash drive, choose “Open folder to view files” and then open the file “start.exe”. If using a DVD, select “Run start.exe”. In the case that the Autoplay dialog does not appear, go to the root of your media and run “start.exe”. Click the UDEC button from the options provided to start the program installation. The installation wizard will guide you through the process. The wizard comprises four screens (Welcome, End User License Agreement, Destination Folder and Ready to Install). In most cases, the user can accept the default options (click Next) presented in these screens.

After installing the software, remove the installation USB drive. Insert the security key in an available USB port. The first time this is done, the operating system will locate the driver that was supplied with the program installation and install it for use with the security key.

Uninstallation

Uninstallation is best performed using the the Windows Programs and Features facility in the Windows Control Panel. To access it, choose Start —> Control Panel —> Uninstall a program. On Windows 8 and later, it is allso possible to right-click on the UDEC program tile and click the “Uninstall” option that appears on the menu at the bottom of the screen to go directly to the “Programs and Features” facility.
