How to Get Help

To open this Help file:
From the main menu, select Help â€”> Help.
To get help with the current dialog or control:
Look for a help button ( ). When it appears, pressing it will bring the user to the page in this Help file that describes the use of the dialog or the control.
To get the definition of the command or FISH function on the current line (in the Editor pane or in the Console pane):
Press the F1 key.
To get a list of keywords that may be used with the current command or keyword when working on the command line in the Console pane:
Type ? after the command word or keyword and press Enter. The console output will show a listing of the valid keywords that may follow the command word/keyword.

If the above resources are not able to provide the assistance you need, consider contacting Itasca for technical support.
