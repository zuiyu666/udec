Finding a License Number

Obtaining your license number is necessary for requesting technical support, upgrading your software, and identifying past purchasing history when making new purchases. The software provides multiple methods for finding your license number. There are variances between the codes, so not every method listed below will work in every case.

  • enter program list security at the command prompt

In the event that the information cannot be obtained using any of the techniques above, the license number is also printed on the tag attached to the USB security key.
