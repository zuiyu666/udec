Get Program Updates

Use the menu sequence Tools —> Check Modifications…. This will open the revision history for the program in a new browser window. The revisions as shown are sorted by date, from newest (top) to oldest (bottom). Compare the latest subversion number (first column) to the one shown in the program’s title bar. A greater subversion number in the revision history indicates a later version than the one you are operating is available. The link above the table containing the revision history provides access to the download page that can be used to access the program update, if you wish to get it.

Program updates are distributed as installation packages. To perform the program update, simply run the downloaded .msi file and follow the prompts to complete installation of the update.

The program’s revision history is also accessible through the action of the Update Alert dialog. See the topic on that dialog for more information.
